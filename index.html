Lightsel

The simpliest carousel for you and your IE8.

  • Installation
  • Options
  • API
  • Example

Short

A light and simple carousel
No additional libraries required
Crossbrowser compatibility. IE8+
13kb of JS code while compressed
Tested on Chrome, Firefox, IE8 and Safari 5.1.4

Features

  • Events
  • Flexible design
  • Infinite spin option
  • Autoscroll with set delay
  • Filling with copies
  • Supports swipes for mobiles

Design

Lightsel has no design.
You design it the way you want.
It requires minimal work and css knowledge.

Options

Name Description Default Values
id Sets a unique identifier for a specific instance of lightsel lightsel1 alphanumeric
arrows If true, the arrows (left, right) will be added, also the "arrows" class will be added to the main element with the given id false true, false
infiniteSpin If true, the user will be able to infinitely spin in both directions. Copies of the elements will be added to the end and to the beginning. The fillWithCopies option will be automatically set to true false true, false
autoscroll If true, lightsel will move the active class to the next item in a set interval (autoscroll_interval) false true, false
autoscroll_interval Sets the interval of autoscrolling 3000 milliseconds
fillWithCopies Fills with copies of the elements until the container is full false true, false
spinOnLessObjects Allows the user to spin the lightsel even if the container is not overflown false true, false
arrowHoldInterval If you hold the arrow buttons, this parameter sets the time of the setInterval function that changes the active object (less time - faster speed) 100 milliseconds
events If set to true, events will pop-up after user actions. true true, false
inertionStartSpeed The speed of the moving part after a drag has been made. It eventually fades down to 0 20 pixels

API

Function Parameters Description
next callback - the function to be called after the method ends
showAsFirst - boolean value that indicates if the next active element should be shown as first in the lightsel 		Moves the active marker to the next element after the current one
previous callback - the function to be called after the method ends
showAsFirst - boolean value that indicates if the next active element should be shown as first in the lightsel 		Moves the active marker to the previous element before the current one
setObjectActive callback - the function to be called after the method ends
newActiveID - the id in the array of object to be set as new active 		Makes the specified object active
getCurrentActiveObject callback(current) - the function to be called after the method ends. Accepts 1 parameter, the return value of the function (current active object)
Returns the array ID of the current active object
showObject callback - the function to be called after the method ends
objectID - the array ID of the object the should be brought to view
showAsFirst - boolean value that indicates whether the object should be shown as the first element in the view 		Brings the object to view (if not visible)
getObjects callback(objectsArray) - the function to be called after the method ends. Accepts 1 parameter, the return value of the function (objects array)
Brings the object to view (if not visible)

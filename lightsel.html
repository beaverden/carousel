Lightsel

The simpliest carousel for you and your IE8.

Short

A light and simple carousel.
Crossbrowser compatibility (IE8+)
Tested on IE8 and Safari 5.1.4 (Windows version).

Features

  • Infinite spin option
  • Autoscroll with set delay
  • Filling with copies
  • Events

Design

Lightsel has no design.
You design the way you want.
It requires minimal work and css knowledge.

Options

Name Description Default Values
id Sets a unique identifier for a specific instance of lightsel lightsel1 alphanumeric
infiniteSpin If true, the user will be able to infinitely spin in both directions false true, false
autoscroll If true, lightsel will move the active class to the next item in a set interval (autoscroll_interval) false true, false
autoscroll_interval Sets the interval of autoscrolling 3000 milliseconds
fillWithCopies Fills with copies of the elements until the container is full false true, false
spinOnLessObjects Allows the user to spin the lightsel even if the container is not overflown false true, false
arrowHoldInterval If you hold the arrow buttons, this parameter sets the time of the setInterval function that changes the active object (less time - faster speed) 100 milliseconds
events If set to true, events will pop-up after user actions. true true, false
inertionStartSpeed The speed of the moving part after a drag has been made 20 pixels
